# RVITM Smart Room Allocation

A simple full-stack prototype for room booking and approval.

## What Is Added

- Flask backend with SQLite database
- Demo role login for Student, Faculty, HOD, and Admin
- Room availability API and frontend
- Booking request form with capacity and time conflict checks
- Faculty/HOD/Admin approval and rejection workflow
- Notifications
- Simple admin analytics
- RVITM-inspired frontend using official room/lab image references
